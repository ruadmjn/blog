How to start

Linux

sudo docker pull ruadmjn/squid3-sarg

Windows

  2. Learn How to use it (can be helpfull for port exposing and forwarding)
  3. Search squid3-sarg
  4. Download

Squid3 monitoring

squid3 starts on docker_ip:3128

Use this ip:port in your host machine to proof squid3.
